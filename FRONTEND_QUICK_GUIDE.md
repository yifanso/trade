## API调用示例

### 使用JavaScript调用下载API

```javascript
// 触发下载
async function downloadStocks(codes) {
    const response = await fetch('/api/trigger-download', {
        method: 'POST',
        headers: {
            'Content-Type': 'application/json'
        },
        body: JSON.stringify({
            stock_codes: codes
        })
    });
    
    const data = await response.json();
    if (data.status === 'success') {
        console.log(`下载完成: ${data.message}`);
    } else {
        console.error(`下载失败: ${data.message}`);
    }
}

// 获取下载状态
async function getDownloadStatus() {
    const response = await fetch('/api/download-status');
    const data = await response.json();
    console.log('统计信息:', data.statistics);
}

// 使用
downloadStocks('000858.SZ, 000651.SZ');
```

### 使用Python调用API

```python
import requests

# 触发下载
response = requests.post(
    'http://localhost:5000/api/trigger-download',
    json={'stock_codes': '000858.SZ, 000651.SZ'}
)

result = response.json()
print(f"状态: {result['status']}")
print(f"消息: {result['message']}")
print(f"详情: {result['details']}")
```

## 常见问题

### Q1: 下载速度很慢
**A**：
- 第一次下载会下载整个历史数据（较慢）
- 后续下载启用增量更新（很快）
- 网络原因也会影响速度

### Q2: 如何查看已下载的数据
**A**：
数据存储在本地：
```bash
~/.cache/qlib_backtest/data/
```

查看文件：
```bash
ls ~/.cache/qlib_backtest/data/
# 示例输出：
# 000858_SZ_data.csv
# 000651_SZ_data.csv
```

### Q3: 按钮点击无反应
**A**：
1. 检查浏览器控制台（F12）是否有错误
2. 检查Flask服务器是否正在运行
3. 确认输入不为空
4. 尝试刷新页面

### Q4: "请指定要下载的股票代码"错误
**A**：
- 输入框为空
- 仅输入了空格或特殊符号
- 请输入有效的股票代码

## 技术细节

### 前端技术栈

- HTML5 + CSS3 + JavaScript
- Fetch API 进行异步通信
- 实时状态刷新（30秒间隔）

### 后端技术栈

- Flask Web框架
- DataDownloader 核心类
- SQLite 数据库记录

### 数据流

```
用户输入
  ↓
前端验证
  ↓
AJAX请求 POST /api/trigger-download
  ↓
后端调用 DataDownloader.download_data()
  ↓
从QLib获取数据
  ↓
保存到本地缓存
  ↓
返回JSON响应
  ↓
前端显示结果
```

## 故障排查快速指南

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 页面无法打开 | Flask未运行 | 检查是否执行了 `python examples/web_frontend.py` |
| 按钮无反应 | JavaScript错误 | 打开浏览器F12查看控制台错误 |
| 下载失败 | 网络问题或QLib问题 | 检查QLib安装和数据路径 |
| 结果为空 | 输入错误 | 确认股票代码格式是否正确 |
