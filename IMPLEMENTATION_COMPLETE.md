## 📋 实现内容

### 1. 新增页面

#### 📥 数据下载管理页面 (`/download-manager`)

**主要功能**：
- 🚀 **立即触发下载** - 输入股票代码并立即下载
- 📊 **下载状态监控** - 显示定时下载器运行状态
- 📈 **统计信息展示** - 显示下载历史和成功率
- 📋 **下载历史列表** - 显示最近的下载记录

**特点**：
- ✅ 支持单只或多只股票
- ✅ 支持增量更新（自动检测上次日期）
- ✅ 支持多分隔符（逗号、中文分号）
- ✅ 实时反馈和加载动画
- ✅ 详细的结果展示

### 2. API 端点

#### POST `/api/trigger-download`
触发数据下载

**请求**：
```json
{
  "stock_codes": "000858.SZ, 000651.SZ"
}
```

**响应**：
```json
{
  "status": "success",
  "message": "成功下载 2/2 只股票",
  "successful": 2,
  "total": 2,
  "details": {
    "000858.SZ": {"records": 2500, "success": true},
    "000651.SZ": {"records": 2300, "success": true}
  }
}
```

#### GET `/api/download-status`
获取下载器状态与统计

**响应**：
```json
{
  "status": "success",
  "running": true,
  "scheduler_active": true,
  "recent_downloads": [...],
  "statistics": {
    "unique_stocks": 10,
    "total_downloads": 100,
    "successful": 98,
    "failed": 2
  }
}
```
